# IHC Accessibility Project

## Sobre o Projeto

O IHC Accessibility Project é um projeto acadêmico desenvolvido para a disciplina de Interação Humano-Computador (IHC).

O objetivo do sistema é promover a inclusão digital por meio de uma interface simples, intuitiva e acessível, voltada principalmente para pessoas com pouca familiaridade com tecnologia, baixa alfabetização digital ou acesso limitado a recursos tecnológicos.

Durante o desenvolvimento, buscamos aplicar conceitos estudados na disciplina de IHC, priorizando a usabilidade, a acessibilidade e a redução de barreiras que dificultam a interação dos usuários com sistemas digitais.

---

## Problema Abordado

Muitas plataformas digitais apresentam interfaces complexas, excesso de informações e fluxos de navegação que podem dificultar o uso por pessoas que:

- Possuem pouca experiência com computadores e internet;
- Têm dificuldades de leitura ou interpretação;
- Não utilizam tecnologia com frequência;
- Necessitam de interfaces mais simples e intuitivas.

Este projeto foi desenvolvido com a proposta de minimizar essas dificuldades e tornar a experiência mais acessível para diferentes perfis de usuários.

---

## Objetivos

- Aplicar conceitos de Interação Humano-Computador (IHC);
- Desenvolver uma interface acessível e intuitiva;
- Promover inclusão digital;
- Facilitar o acesso à informação por usuários com pouca experiência tecnológica;
- Explorar recursos de inteligência artificial como apoio à experiência do usuário.
